Core Grant & Facility Programs

🎓 Student Transport Grant

  • Amount 50 OMR/month
  • Trigger Enrollment Verified
  • Frequency Monthly Batch
  • Approval Auto + Spot-Check

👶 Newborn Baby Facility

  • Trigger Birth Event (FHIR)
  • Linkage Parent → Household
  • Frequency Event-Driven
  • Approval Auto / Manual Review

🍞 BPL Food Support

  • Eligibility Poverty Index < 0.4
  • Segments Elderly · Disabled · Large HH
  • Frequency Monthly Allocation
  • Approval Policy Rule + HITL

🏢 CSR Fund Allocation

  • Input Budget + Arabic Query
  • AI Action Segment Ranking via RAG
  • Output Ranked Options + Justify
  • Approval Mandatory Human Review
