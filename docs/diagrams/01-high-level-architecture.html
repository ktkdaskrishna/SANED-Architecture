🏛️ Government of Oman — Social Protection Fund

PASI Platform — High-Level Architecture

Data Lakehouse · MLOps · AI Agentic Platform — 8 Logical Layers

Layer 1
Source Integration
Ministry & external data ingestion
🏢 Civil Identity API NCSI civil registry, family records, national ID
🎓 Education Systems Student enrollment, attendance, institutions
💼 Labor & Employment Employment status, wage records, job registries
🏥 Health Systems Disability, birth/death events, medical indicators
📍 Geography & Address Municipal records, address hierarchy
💰 Finance & Payments Benefit disbursement, CSR allocation records
📋 Batch File Drops CSV, Excel, JSON file uploads via secure SFTP
🔌 Event Streams Kafka/Event Hub for real-time events (births, deaths)
Layer 2
Data Lake Storage
Raw → Standardized → Curated zones
📦 Raw Landing Zone Immutable raw ingestion — forensic traceability
🔄 Standardized Zone Schema-enforced, validated, deduped records
⭐ Curated / Gold Zone Analytics-ready, governed marts & feature stores
🗄️ Object Storage (S3/ADLS) Delta Lake / Iceberg on cloud object store
📊 Metadata Catalog Unity Catalog / Glue — lineage, classification, governance
🔒 Quarantine Zone Failed-validation records for stewardship review
Layer 3
Processing & ETL
Spark-based transformation pipelines
⚡ Apache Spark Engine Distributed compute for batch & streaming ETL
🧹 Data Quality Engine Great Expectations / Soda — schema, freshness, plausibility
🔗 Entity Resolution Deterministic + probabilistic matching across sources
👨‍👩‍👧‍👦 Household Builder Family linkage, dependency, residence-based construction
🔀 CDC Pipeline Change Data Capture for incremental processing
📐 Schema Evolution Automated schema migration & versioning
Layer 4
Integrated Registry
Golden records & historized entities
👤 Person Registry Golden person records with SCD Type-2 history
🏠 Household Registry Composed from family, residence, dependency links
🏫 Institution Registry Schools, hospitals, CSR partners, capacity data
📜 Benefit Programs Program definitions, versions, active windows
✅ Eligibility Decisions Decision log: eligible, rejected, pending, appealed
📅 Life Events Birth, death, marriage, enrollment, employment changes
🌍 Geography Hierarchy Governorate → Wilayat → Area → Block mapping
Layer 5
Rules & ML Engine
Eligibility, scoring, MLOps
⚖️ Rules Engine No-code eligibility rules, Boolean logic, thresholds
🎯 Segment Builder Reusable population segments: students, elderly, etc.
🧠 ML Training Pipelines Clustering, ranking, anomaly detection, proxy scoring
📦 Model Registry MLflow — version, stage, metrics, governance approval
🔍 Feature Store Reusable features: income proxy, vulnerability score, etc.
📈 Model Monitoring Drift detection, performance tracking, alert triggers
🧪 Simulation Engine Policy impact simulation before production rollout
Layer 6
Analytics & Serving
Dashboards, APIs, BI marts
📊 SQL Serving Layer Governed analytical warehouse / SQL endpoint
📈 BI Dashboards PowerBI / Superset — enrollment, coverage, budget
🔌 REST/GraphQL APIs Programmatic access for downstream consumers
📋 Reporting Engine Scheduled reports: regional, program overlap, CSR
📡 Event Publisher Push notifications for eligibility changes, alerts
Layer 7
AI Agentic Platform
Conversational AI & agent workflows
🤖 AI Orchestrator LangGraph / Semantic Kernel — multi-agent coordination
💬 Arabic/English Chat Bilingual conversational interface for authorized users
🔧 Tool Registry Approved tools: eligibility lookup, segment query, allocation
🛡️ AI Guardrails Prompt filtering, PII masking, governed data access
🏗️ Agent Builder Admin-configured agents with human-in-the-loop approval
📝 Explanation Engine Decision justification, confidence scores, traceability
Layer 8
Security & Governance
Cross-cutting: IAM, encryption, audit
🔐 IAM & RBAC/ABAC SSO, MFA, role-based + attribute-based access
🔑 Key Management Organization-managed keys, rotation, audit logging
🎭 Data Masking Tokenization, column-level masking for PII
📜 Audit Trail Immutable logs: every action, actor, timestamp
📡 Observability Prometheus + Grafana — pipeline health, latency, drift
🛡️ SIEM Integration Security events forwarded to SOC / Splunk
📋 Compliance Engine Data residency, retention, consent enforcement
