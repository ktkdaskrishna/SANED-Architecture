Security Architecture

Security & Governance Framework

Defense-in-depth controls across identity, encryption, data protection, AI safety, audit, and compliance

🔐

Identity & Access Management

  • ✓ Enterprise IDAM for MoF/Internal with SSO Federation
  • ✓ High-Scale CIAM for 1.2M+ Citizens (Biometric Auth)
  • ✓ MFA enforced for all actors (e-ID / TOTP)
  • ✓ RBAC for operations + ABAC for data-scoped access
  • ✓ Core Service identities for agents & integrations
  • ✓ Quarterly access reviews & auto-revocation loops
🔑

Encryption & Key Management

  • ✓ TLS 1.3 in transit with strict cipher suites
  • ✓ AES-256 at rest across all storage tiers
  • ✓ FIPS 140-2 L3 Hardware Security Modules (HSM)
  • ✓ Segregated key domains (Raw, Curated, Secrets)
  • ✓ Automated HSM-backed Key Rotation Policy
  • ✓ All cryptographic usage deeply audited
🎭

Data Protection & Privacy

  • ✓ National ID tokenization in lower-trust contexts
  • ✓ Column-level masking for sensitive attributes
  • ✓ Row-level security by role, geography, program
  • ✓ Data classification: Public, Internal, Confidential, Restricted
  • ✓ Consent and sharing constraints enforcement
  • ✓ Data residency: all data within Oman jurisdiction
🛡️

AI Safety & Guardrails

  • ✓ No unrestricted model access to raw PII
  • ✓ Tool-use restricted to approved governed connectors
  • ✓ Prompt injection and jailbreak detection
  • ✓ PII masking in prompts and responses
  • ✓ Complete prompt/tool/output audit trail
  • ✓ Human approval for high-impact AI outputs
📜

Audit & Observability

  • ✓ Immutable ledger logs for all material actions
  • ✓ Imperative Database Activity Monitoring (DAM)
  • ✓ SIEM Integration with deep Pipeline Observability
  • ✓ Immediate Anomaly detection on Data Exfiltration
  • ✓ Traceability metrics for full Lakehouse SQL queries
  • ✓ Executive Compliance and SLA Operations Dashboards
⚖️

Compliance & Governance

  • ✓ Data retention policies per classification
  • ✓ Segregation of duties enforcement
  • ✓ Data lineage and provenance tracking
  • ✓ Model governance: approval gate before production
  • ✓ Schema change approval workflows
  • ✓ Regular compliance audits and reporting
